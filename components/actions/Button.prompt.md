Buttons drive every action; the primary (brass) variant marks the single most important step on a view.

```jsx
<Button variant="primary" arrow>Start a conversation</Button>
<Button variant="outline" size="sm">View services</Button>
```

Variants: `primary` (brass, one per view), `secondary` (solid ink), `outline` (hairline), `ghost` (quiet). Sizes: `sm` / `md` / `lg`. Set `arrow` for the forward-nudge CTA, `as="a"` + `href` for links, `iconLeft` for a leading glyph.
